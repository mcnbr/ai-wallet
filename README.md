# Carteira - Investment Tracking Application

A comprehensive investment tracking application for stocks, FIIs, ETFs, Tesouro Direto, Fixed Income, Crypto, and Stocks.

## Tech Stack
- **Frontend**: Next.js 14+ (App Router), TypeScript, Tailwind CSS
- **Backend**: FastAPI (Python 3.11+), PostgreSQL, TimescaleDB
- **Data Sources**: yfinance, CoinGecko, Tesouro Direto API

## Features
- multi-asset support (Stocs, UI, FIIs, Crypto, etc.)
- Real-time price tracking
- Historical performance analysis
- Modern, Premium UI/UX with Dark Mode
- Secure authentication
